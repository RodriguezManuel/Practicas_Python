##  Cómo instalar las dependencias (Requisito)

Para ejecutar este proyecto, es necesario instalar **JupyterLab** en un entorno virtual aislado. Los pasos exactos para hacerlo desde la consola son:

1. **Crear el entorno virtual** en la carpeta principal del proyecto:
   
   py -m venv venv

2. **Activa el Entorno Virtual**
    
    .Si usas Windows: .\venv\Scripts\activate
    .Si usas Linux/Mac: source venv/bin/activate

3. **Instalar las dependencias**

    pip install -r requirements.txt

4. Una vez instaladas las dependencias, inicia Jupyter Lab con el siguiente comando en consola:

    Jupyter lab
